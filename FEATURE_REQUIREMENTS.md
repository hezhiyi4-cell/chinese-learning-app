
# 功能需求详细文档

## MVP 版本（阶段1-3）

### 1. 用户系统 ✅
- [ ] 邮箱注册/登录
- [ ] 个人资料编辑
- [ ] 头像上传
- [ ] 多语言界面切换（英文、日文、韩文、法文、西班牙文、俄文）

### 2. 课程系统
- [ ] 5级分级课程（L0-L4）
  - **L0**: 拼音入门、声调学习
  - **L1**: 日常单词、简单短句
  - **L2**: 日常对话、简单造句
  - **L3**: 完整句式、场景对话
  - **L4**: 语法进阶、短文阅读、商务中文
- [ ] 课时内容（文字、音频、视频）
- [ ] 词汇卡片
- [ ] 学习进度追踪

### 3. 练习系统
- [ ] 听音辨声调
- [ ] 词汇选择
- [ ] 句子跟读
- [ ] 填空题
- [ ] 语法练习

### 4. AI 功能
- [ ] 语音识别（Whisper）
- [ ] 声调纠错
- [ ] 发音评分
- [ ] AI 助教对话
- [ ] 语法纠错

### 5. 游戏化系统
- [ ] 闯关模式
- [ ] 积分系统
- [ ] 徽章系统
- [ ] 排行榜
- [ ] 连续打卡奖励

## 开发优先级

### P0 - 必须有（MVP）
1. 用户注册登录
2. 基础课程（L0-L1）
3. 简单练习
4. 基础UI

### P1 - 应该有
1. AI 语音识别
2. 声调纠错
3. 积分系统
4. 徽章

### P2 - 可以有
1. 完整课程体系
2. 社群功能
3. 排行榜
4. 多语言完整支持
