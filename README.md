
# 外国人学中文 - 多端游戏化学习APP

## 项目简介
这是一个面向外国人的中文学习平台，结合游戏化学习体验，支持iPad、电脑端多端使用。

## 项目架构
- **前端**: Flutter（一套代码多端适配）
- **后端**: Golang + Gin（高性能API）
- **数据库**: PostgreSQL + Redis
- **AI**: OpenAI Whisper + GPT-4o-mini
