
# 📱 Flutter 安装和启动 - 简易指南（中文版）

## 🎯 超简单安装方法（按步骤做即可）

---

## 🚀 方法一：使用我为你准备的安装脚本（最简单）

### 第一步：右键运行安装脚本

1. 找到桌面 `ChineseLearningApp` 文件夹
2. 找到 `install_flutter.ps1` 文件
3. **右键点击** → 选择 **"使用 PowerShell 运行"**

如果出现安全警告，选择 **"打开"**

### 第二步：等待安装完成

脚本会自动：
- ✅ 下载 Flutter SDK
- ✅ 解压到 C:\flutter
- ✅ 配置环境变量
- ✅ 配置国内镜像（下载更快）

### 第三步：关闭并重新打开终端

安装完成后，**关闭所有 PowerShell/命令提示符窗口**

## 🚀 方法二：手动安装（如果方法一不行）

### 第一步：下载 Flutter SDK

1. 访问这个网页：https://docs.flutter.dev/release/archive?tab=windows
2. 下载 **Stable** 版本（稳定版）
3. 下载的文件是一个 zip 压缩包

### 第二步：解压文件

1. 把下载的 zip 文件解压到 **C:\flutter**
2. 确保解压后你能看到这个路径：
   ```
   C:\flutter\bin\flutter.bat
   ```

### 第三步：配置环境变量

1. 在 Windows 搜索栏输入 **"环境变量"**
2. 点击 **"编辑系统环境变量"**
3. 点击 **"环境变量"** 按钮
4. 在 **"用户变量"** 区域，找到 **"Path"**，点击 **"编辑"**
5. 点击 **"新建"**，输入：
   ```
   C:\flutter\bin
   ```
6. 点击 **"确定"** → **"确定"** → **"确定"**

### 第四步：配置国内镜像（可选，但推荐）

1. 同样打开环境变量设置
2. 在 **"用户变量"** 区域，点击 **"新建"**
3. 第一个：
   - 变量名：`PUB_HOSTED_URL`
   - 变量值：`https://pub.flutter-io.cn`
4. 第二个：
   - 变量名：`FLUTTER_STORAGE_BASE_URL`
   - 变量值：`https://storage.flutter-io.cn`
5. 点击 **"确定"**

## ✅ 验证 Flutter 是否安装成功

1. **关闭所有终端**（重要！）
2. 打开一个 **新的** PowerShell 窗口
3. 输入：
   ```bash
   flutter --version
   ```
4. 如果你看到类似这样的输出，说明安装成功了！
   ```
   Flutter 3.22.0
   ```

---

## 🎮 启动项目！

Flutter 安装成功后，按以下步骤启动：

### 第一步：进入项目文件夹

在 PowerShell 中运行：
```bash
cd c:\Users\user\Desktop\ChineseLearningApp\ai_trae_app
```

### 第二步：安装依赖

运行：
```bash
flutter pub get
```

### 第三步：启动应用（Chrome浏览器）

运行：
```bash
flutter run -d chrome
```

## ⚠️ 常见问题

### 问题 1：运行 `flutter --version` 提示找不到命令
**原因**：环境变量没生效
**解决**：关闭所有终端窗口，重新打开

### 问题 2：`flutter pub get` 下载很慢
**原因**：没配置国内镜像
**解决**：按照上面的方法配置国内镜像

### 问题 3：提示连接后端失败
**原因**：后端没启动
**解决**：先运行后端：
```bash
cd c:\Users\user\Desktop\ChineseLearningApp\backend
go run ./cmd/server
```
